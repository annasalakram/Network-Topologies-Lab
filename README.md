
# Network Topologies Lab 🌐

Welcome to my Network Engineering portfolio. This repository is a collection of various network topology designs, configurations, and simulations. It serves as a documentation of my journey in mastering Network Administration and Infrastructure.

## 📌 Project Overview
The goal of this repository is to demonstrate proficiency in:
- **Routing & Switching** (VLANs, OSPF, EIGRP, BGP, STP)
- **Network Security** (ACLs, NAT, VPN, Port Security)
- **Infrastructure Design** (Scalable LAN/WAN architectures)
- **Network Automation** (Future roadmap)

## 🛠️ Tools Used
- **Simulators:** Cisco Packet Tracer, GNS3, PNETLab.
- **Hardware (Emulated):** Cisco ISR Routers, Catalyst Switches.
- **Documentation:** Draw.io for network diagrams, Markdown for lab reports.

---

## 🚀 How to Navigate This Repo
1.  Navigate to a specific project folder (e.g., `/basic-vlan-routing`).
2.  Read the `lab-scenario.md` for detailed objectives and addressing tables.
3.  Check the `/config-files` directory to see the CLI configurations for each device.
4.  Download the `.pkt` or `.gns3` file to run the simulation on your local machine.
